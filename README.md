# Retail Sales Analytics Dashboard

A Business Intelligence project built using **Excel, SQL (SQLite), and Power BI** to analyze retail sales performance and generate actionable business insights through interactive dashboards.

---

## Project Overview

This project demonstrates an end-to-end analytics workflow, starting from data cleaning to dashboard creation.

The objective was to analyze retail sales data, evaluate business performance, identify customer purchasing patterns, and visualize key performance indicators (KPIs) that support data-driven decision making.

---

## Objectives

- Clean and preprocess raw retail sales data.
- Perform business analysis using SQL.
- Validate key business metrics.
- Build an interactive dashboard in Power BI.
- Generate meaningful business insights.

---

## Tech Stack

- Microsoft Excel
- SQL (SQLite)
- Power BI
- DAX

---

## Project Workflow

1. Cleaned the raw retail dataset using Microsoft Excel.
2. Imported the cleaned dataset into SQLite.
3. Performed SQL queries to analyze business performance and validate KPIs.
4. Imported the cleaned data into Power BI.
5. Created DAX measures for business metrics.
6. Designed an interactive dashboard with slicers and charts.

---

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin (%)

## Business Insights

- Technology generated the highest overall sales.
- Consumer segment contributed the largest share of revenue.
- West region recorded the highest sales performance.
- Standard Class was the most frequently used shipping mode.
- Sales showed a positive growth trend over multiple years.
